# 一、JavaScript入门
## 1、JavaScript与HTML和CSS的协作工作
- #HTML 提供结构、是一种标记语言
- #CSS 增加样式、是样式表语言
- #JavaScript 实现行为、是一种编程语言
- 提供网页交互效果、动态更改内容
## 2、数据类型
- #Number 整数和浮点数
- #String 字符或文本组成的序列
- #Boolean true or false
- #undefined 已被声明但未被赋值
- #null 变量值被故意设置为“无”
- #Object 键值对的集合
- #Symbol 为属性创建唯一的标签或标识符
- #BigInt 用于超出 #Number 类型限制的超大数字
	- 在大数字末尾添加**n**来创建 #BigInt 
## 3、变量的概念
- #let 声明且允许后续修改数值的变量
- #const 被声明并赋值后变量就不可修改
- #驼峰命名法 第一个单词全部小写，后续每个单词的首字母都大写
```js
let thisIsCamelCase; //for example 
```
# 二、字符串介绍
## 1、字符串的两个特性
- 可以使用单引号或单引号
>引号使用必须保持一致性
- 不可变性
>即当将变量赋新值时， 会创建新的字符串
### 2、字符串拼接
- 使用加法连接
```JS
let firstname = "moe";
let lastname = "form";
// 先行定义firstname和lastname
let fullname = firstname + " " + lastname;
console.log(fullname);
// "moeform"
```
- 使用`+=``
```JS
fullname += ' OK';
console.log(fullname);
// "moeform OK"
```
- 使用`concat()`
```JS
let updateText = "letgo";
let fullnameNext = fullname.concat('' + updateText + 'end');
console.log(fullnameNext);
// "moeform OK letgo end"
```
### 3、`console.log()`可用逗号传递
```JS
let name = "Alice";
let age = 25;
console.log("Name:", name, "Age:", age); // Name: Alice Age: 25
```
